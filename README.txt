## AlxMayor

Plugin para WooCommerce que permite establecer un monto mínimo de compra.
Y otras opciones de configuración para tiendas mayoristas.

### Versión: 1.0.5

Instalación:
1. Descomprimir el archivo y subirlo a la carpeta wp-content/plugins/alxmayor
2. Activar el plugin desde el panel de administración de WordPress
3. Configurar las opciones del plugin en la sección "AlxMayor" 


requerimientos:
- WooCommerce
- PHP 7.4 o superior
- WordPress 5.0 o superior
